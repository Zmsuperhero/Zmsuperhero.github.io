如何从零部署一个博客

  1. 拥有一个GitHub账号。 为了更愉快的冲浪，可以装个Google插件或者挂个梯子或者怎么地都行
  2. 了解GitHub库的概念以及hexo框架
  3. 以上两步均可忽略，最重要的是找到一个适合自己的教程，如果你一点都不了解相关知识，请尽量看视频，比如这个视频
  4. 哈哈哈哈哈哈哈哈本篇内容到此结束

搭建时可能出现的问题

  1. 不要用版本太低的hexo和node

  2. hexo d之后出现 error: failed to push some refs to 相关报错
    解决方式：另起一行run两下

    git config –global user.email 你的邮箱
    git config –global user.name 你的名字

  3. hexo d报错，多半是网不好，不要慌，多刷新几次，总能同步上的

  4. 注意空格和英文
